# WanAndroid

一个基于 Jetpack Compose 开发的 WanAndroid 客户端应用。

## 项目简介

本项目使用 [WanAndroid API](https://www.wanandroid.com/) 提供的开放接口，采用现代化的 Jetpack Compose 技术栈进行开发。

## 技术栈

- **Kotlin** - 开发语言
- **Jetpack Compose** - 现代化 UI 框架
- **Material Design 3** - UI 设计规范
- **MVVM** - 架构模式
- **Retrofit** - 网络请求
- **OkHttp** - HTTP 客户端
- **Gson** - JSON 解析
- **Kotlin Coroutines** - 异步处理
- **Coil** - 图片加载

## 功能模块

- 首页
- 项目
- 公众号
- 体系
- 我的

## 开发环境

- Android Studio Hedgehog | 2023.1.1+
- Kotlin 2.0.21
- Gradle 8.11.1
- Min SDK: 24
- Target SDK: 36
- Compile SDK: 36
